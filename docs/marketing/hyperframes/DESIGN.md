# cc-clip Promo Video Design

## Style Prompt

Dark, precise developer-tool launch video for an SSH clipboard bridge. The
visual language is grounded in the existing README accent color and terminal
demo assets: off-black canvas, warm amber transport lines, restrained teal
status detail, crisp terminal glass, and honest open-source utility rather than
big-product spectacle.

## Colors

- Canvas: `#11100E`
- Ink: `#F7F0E6`
- Muted ink: `#B7AFA3`
- SSH amber: `#D97706`
- Ready teal: `#14B8A6`
- Panel graphite: `#1D1B18`

## Typography

- Display and labels: system sans stack, led by `SF Pro Display` and `Inter`
  where available.
- Code and terminal details: `JetBrains Mono`, `SFMono-Regular`, `Menlo`, or
  the platform monospace fallback.

## Motion

- Precise entries with `power3.out` and short 0.35-0.7s moves.
- Light-path and status motion are ambient, finite, and subtle.
- Scene transitions use fades, slides, and scale changes rather than decorative
  shader effects.

## What NOT To Do

- Do not imply official integration with Anthropic, OpenAI, opencode, or Google.
- Do not present Antigravity as image-paste support.
- Do not describe the tool as universal clipboard sync.
- Do not use neon cyberpunk, robots, or generic AI spectacle.
- Do not put stale beta install commands in stable `v0.9.0` demo material.
